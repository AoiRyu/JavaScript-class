Other Ways to Create Arrays

Creating Arrays

Syntax:
var courses = new Array(3);
courses[0] = 'JS';
courses[1] = 'HTML';
courses[2] = 'PHP';

Creating Arrays Part. 2

Syntax:
var courses = new Array();
courses[0] = 'JS';
courses[1] = 'HTML';
courses[2] = 'PHP';
courses[3] = 'CSS';

or

Syntax:
var courses = ['HTML','JS','CSS','PHP' ];

Arrays Properties & Methods

The Length Property

Syntax:
var courses = ['HTML','JS','CSS','PHP'];
document.write(courses.length);
//outputs 4

Combining Arrays

Syntax:
var c1 = ['HTML','CSS'];
var c2 = ['JAVA','C++'];
var courses = c1.concat(c2);

The courses array that results contains 4 elementes (HTML, CSS, JAVA, C++)

Math Objects Methods

Property Description
abs(x) Returns the absolute value of x
acos(x) Returns the arccosine of x, in radians
asin(x) Returns the arcsine of x, in radians
atan(x) Returns the arctangent of x as a numeric value between - PI/2 and PI/2 radians
atan2(x,y) Returns the arctangent of the quotient of its arguments
ceil(x) Returns x, rounded upwards to the nearest integer
cos(x) Returns the cosine of x (x is in radians)
exp(x) Returns the value of E^x
floor(x) Returns x, rounded downwards to the nearest integer
log(x) Returns the natural logarithm (base E) of x
max(x,y,z,w...n) Returns the number with the highest value
min(x,y,z,w...n) Returns the number with the lowest value
pow(x,y) Returns the value of x to the power of y
random(x) Returns a random number between 0 and 1
round(x) Round x to the nearest integer
sin(x) Returns the sine of x (x is int radians)
sqrt(x) Returns the square root of x
tan(x) Returns the tangent of an angle
